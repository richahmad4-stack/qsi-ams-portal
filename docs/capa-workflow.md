# CAPA workflow

This phase adds controlled CAPA handling inside each audit-stage file.

## Where to use it

Open a client certification file, choose an audit stage, then open **NCR / CAPA**.

Each audit stage can now:

- show NCRs raised during that stage
- create a CAPA from an NCR
- record immediate correction, root cause, corrective action, preventive action, responsible person, and target date
- close CAPA with verification, effectiveness, closure notes, and closure date
- generate a dedicated **NCR / CAPA PDF**

CAPA records remain linked to the NCR and audit event, so Stage 1, Stage 2, Surveillance 1, Surveillance 2, and Recertification keep separate NCR/CAPA histories.
